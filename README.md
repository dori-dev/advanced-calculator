# Install PyQt5

## Windows

```
pip3 install PyQt5
```

## Linux

```
pip3 install PyQt5
sudo apt install python3-pyqt5
sudo apt install pyqt5-dev-tools
sudo apt install qttools5-dev-tools
```

## Mac

```
pip3 install PyQt5
brew install pyqt5
```

#

# How To Run

Download Repository.

```
git clone https://github.com/dori-dev/advanced-calculator.git
cd advanced-calculator
```

Run Python File.

```
python3 main.py
```
